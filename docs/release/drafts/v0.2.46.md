## 한눈에 보기
- 이번 버전은 **WinUI 사용 중 바로 체감되는 작은 마찰과 지원용 진단 혼선을 줄이는 유지보수 업데이트**입니다.
- `Raw Data` 탭에서 긴 WCT/Report 텍스트를 스크롤로 직접 확인하기 쉬워졌습니다.
- `Triage` 탭에서 `Evidence` 패널을 열고 닫을 때 페이지 폭이 흔들리던 현상을 줄였습니다.
- `address_db` 로딩 실패는 이제 파일 누락과 게임 버전 미지원을 구분해서 표시합니다.

## 핵심 변경
- `WCT JSON` 텍스트 박스에 가로/세로 스크롤바를 명시해 긴 JSON을 한 줄 기준으로 직접 탐색할 수 있게 했습니다.
- `Report` 텍스트 박스에 안정적인 세로 스크롤바를 노출해 긴 리포트 탐색성을 개선했습니다.
- 루트 스크롤 영역이 세로 스크롤바 폭을 항상 예약하도록 조정해 `Evidence` expander 토글 시 Triage 레이아웃이 좌우로 흔들리지 않게 했습니다.

## 엔진 / 지원성
- `address_db/skyrimse_functions.json` 로딩 실패를 `파일 없음`과 `현재 game_version 항목 없음`으로 분리해, 사용자가 설치 경로 문제인지 지원 버전 문제인지 더 빨리 판단할 수 있게 했습니다.
- AddressResolver에 load-status 분기를 추가해 진단 문구가 실제 실패 원인과 더 가깝게 맞도록 정리했습니다.

## 빌드 / 운영
- 이번 변경은 UI/XAML 가드 테스트와 AddressResolver 런타임 테스트를 함께 확장해 회귀를 고정했습니다.
- 배포 zip은 `Tullius_ctd_loger_v0.2.46.zip` 형식을 사용합니다.

## 주의사항
- `address_db` 메시지가 더 구체적으로 바뀌어도, 함수 DB 자체는 여전히 포함된 게임 버전 항목 범위 안에서만 동작합니다.
- `msdia140.dll` 경고 동작 자체를 바꾼 것은 아니며, 이번 릴리즈는 주로 UI 안정성과 진단 문구 명확화에 집중합니다.
